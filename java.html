Програмування мовою Java

Java більше 20 років - і це все ще провідна мова програмування. Згідно зі статистикою,більше 70% вакансій для IT-розробників вимагають знання Java. На Java працює понад три мільйони гаджетів - від смартфонів і кавоварок до систем "розумний будинок" і електрокарів. Навчання програмуванню у гуртку відбувається, починаючи з вивчення основних понять і принципів об'єктно-орієнтованого програмування, класів, а також особливостей мови Java. На заняттях гуртка діти дізнаються, що потрібно, для того щоб стати хорошим програмістом і отримають базові знання і навички для досягнення цієї мети. За період навчання гуртківці:
  • дізнаються, як створюються комп'ютерні програми,
  • отримають уявлення про Java-технології для взаємодії з СУБД
  • навчиться створювати прості програми на Java.
На кожному занятті розглядаються приклади, які роблять викладений матеріал дуже наочним і допомагають його кращому засвоєнню.
